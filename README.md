## 📖 About LumiBooks

LumiBooks is a modern full-stack online bookstore designed to provide a seamless digital reading experience. The application enables users to discover books, browse categories, explore trending titles, read books online, and manage their personal library through a clean and responsive interface.

Built with a scalable frontend and backend architecture, LumiBooks demonstrates production-ready development practices, including secure authentication, RESTful APIs, cloud media integration, responsive design, and modular code organization.

The project focuses on delivering a fast, intuitive, and visually polished experience while maintaining clean architecture and maintainable code.

# ⚙️ Getting Started

## Prerequisites

Before running the project locally, make sure you have installed:

- Node.js
- npm
- MongoDB (or MongoDB Atlas)
- Git

---

## Installation

Clone the repository.

```bash
git clone https://github.com/Shrushti2003/Online-Book-Store.git
```

Move into the project directory.

```bash
cd Online-Book-Store
```

Install dependencies.

### Frontend

```bash
cd Frontend
npm install
```

### Backend

```bash
cd Backend
npm install
```

---

## Environment Variables

Create a `.env` file inside the appropriate project directories and configure the required environment variables before starting the application.

Example:

```env
MONGODB_URI=your_database_url
JWT_SECRET=your_secret_key
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
CLERK_SECRET_KEY=your_clerk_secret
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_publishable_key
```

---

## Run Locally

### Backend

```bash
npm run dev
```

### Frontend

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
